## Огляд

**PetFamily** - це сучасна платформа для знаходження свого ідеального домашнього улюбленця. Спеціально розроблена для людей, які почувають себе самотньо і шукають нового друга серед котиків 🐱, собачок 🐶 або екзотичних тварин 🐰.

## ✨ Основні особливості

### 🛍️ Каталог тварин
- **3 категорії**: Кішки, Собаки, Екзотичні тварини
- **10+ різних порід** в кожній категорії
- **Фільтрація за**: породою, статтю, віком, колором
- **Сортування**: за ім'ям, ціною, віком
- Красивий дизайн з емодзі для кожної тварини

### 💳 Система замовлення
- Вибір домашнього улюбленця та перегляд деталей
- Форма замовлення з контактною інформацією
- Контактні дані замовника (ім'я, телефон, email, адреса)
- **Оплата**: під час підтвердження дзвінком (реалістичний спосіб)
- Номер замовлення та статус відслідковування

### 👤 Система авторизації та реєстрації
- Реєстрація нового користувача з валідацією
- Авторизація з сесіями
- Профіль користувача з інформацією
- Вихід з акаунту
- Захист замовлень (доступна тільки авторизованим)

### 📊 Мої замовлення
- Переглід всіх своїх замовлень
- Статси замовлень (очікує, підтверджено, завершено)
- Детальна інформація про кожне замовлення

### 🎨 Премиум дизайн
- Темна тема з градієнтами
- Live Canvas анімація на фоні
- Smooth переходи та анімації
- Glassmorphism ефекти
- Responsive дизайн (mобіль, планшет, ПК)
- **60 FPS анімації**

## 📖 Як використовувати

### Головна сторінка
- Вітання для самотніх людей 💙
- Інформація про сервіс
- Кнопка для переходу до каталогу

### Реєстрація (потрібна для замовлення!)
1. Натисни "Реєстрація"
2. Введи ім'я користувача (мінімум 3 символи)
3. Введи email
4. Введи пароль (мінімум 6 символів)
5. Натисни "Реєструватись"

### Вхід
1. Введи ім'я користувача
2. Введи пароль
3. Натисни "Увійти"

### Пошук тварини
1. Перейди в "Каталог"
2. Вибери вид тварини (Кішки/Собаки/Екзотичні)
3. Використовуй фільтри:
   - **Порода**: напиши назву породи
   - **Стать**: виберіхлопчика або дівчинку
   - **Вік**: виберіз року від 1 до 4 років
   - **Колір**: напиши колір шерсті
   - **Сортування**: за ім'ям, ціною або віком
4. Натисни на тварину для отримання детальної інформації

### Замовлення тварини
1. Натисни "Деталі & Замовити"
2. Заповни форму контактної інформації:
   - Повне ім'я
   - Номер телефону
   - Email
   - Адреса доставки
3. Натисни "Підтвердити замовлення"
4. Чекай дзвінку від менеджера для підтвердження оплати

### Переглід замовлень
1. Перейди в "Мої Замовлення"
2. Переглід всіх замовлень зі статусом
3. Статуси: ⏳ Очікує → ✓ Підтверджено → 🎉 Завершено

## 🛒 Цінообразування

Кожна тварина має свою ціну:
- **Кішки**: від ₴1500 до ₴3000
- **Собаки**: від ₴1500 до ₴3000
- **Екзотичні**: від ₴800 до ₴5000
- **Доставка**: ₴100 (фіксована)

## 🧬 База даних (залік)

Вся інформація зберігається в оперативній пам'яті (симуляція БД):
- **Користувачі**: `users_db` в `app.py`
- **Тварини**: `pets_database` в `app.py`
- **Замовлення**: `orders_db` в `app.py`

> ℹ️ Для справжнього проекту потрібна реальна БД (PostgreSQL/MongoDB)

## 🔧 Кастомізація

### Додати нову тварину
Відкрий `app.py`, знайди `pets_database` і додай:
```python
{
    'id': 7,
    'name': 'Нове ім\'я',
    'breed': 'Нова порода',
    'age': 2,
    'gender': 'М',  # або 'Ж'
    'price': 2000,
    'color': 'Колір',
    'image': 'photo.jpg',
    'description': 'Опис тварини'
}
```

### Змінити кольори
Відкрий `static/style.css` і спрости CSS змінні в `:root`:
```css
:root {
    --primary-color: #FF6B6B;    /* червоний / основний */
    --secondary-color: #4ECDC4;  /* бірюзовий / додатковий */
    --accent-color: #45B7D1;     /* синій / акцент */
}
```
